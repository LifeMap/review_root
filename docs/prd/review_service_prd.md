## 1. 제품 개요

### 1.1 제품 비전

**"흩어진 리뷰를 한곳에 모아, 진짜 리뷰로 구매 결정을 돕는다"**

여러 커머스 플랫폼에 분산된 리뷰를 통합 수집하고, 광고성 리뷰를 필터링하여 실제 구매에 도움이 되는 진성 리뷰를 제공하는 서비스입니다.

### 1.2 핵심 가치

| 가치 | 설명 |
|------|------|
| **리뷰 통합** | 여러 플랫폼에 흩어진 리뷰를 한 곳에서 조회 |
| **진성 리뷰 큐레이션** | 광고성 리뷰 필터링, 단점을 상세히 기술한 리뷰 우선 노출 |
| **구매 전환 지원** | 플랫폼별 가격 비교 및 구매 링크 제공 |
| **카테고리별 맞춤 개인화** | 분야에 따른 개인화 기준 적용 (패션: 체형, 뷰티: 피부타입 등) |

### 1.3 배경 및 문제 정의

#### 현재 사용자가 겪는 문제

| 문제 | 상세 설명 | 영향 |
|------|----------|------|
| 리뷰 분산 | 동일 상품이 여러 플랫폼에 존재하지만 각 플랫폼의 리뷰를 따로 확인해야 함 | 시간 낭비, 정보 누락 |
| 광고성 리뷰 범람 | 협찬/광고 리뷰가 섞여 있어 진짜 후기를 찾기 어려움 | 잘못된 구매 결정 |
| 장점 위주 리뷰 | 대부분의 리뷰가 장점만 나열하여 실제 단점 파악이 어려움 | 구매 후 후회 |
| 정보 과부하 | 수백 개의 리뷰 중 나에게 의미 있는 리뷰를 찾기 어려움 | 결정 피로 |
| 가격 비교 번거로움 | 플랫폼별 가격을 따로 확인해야 함 | 최저가 놓침 |

#### 솔루션

| 솔루션 | 설명 |
|--------|------|
| 멀티 플랫폼 리뷰 통합 | 여러 쇼핑몰의 리뷰를 자동으로 수집하여 한 화면에 표시 |
| 광고성 리뷰 필터링 | AI 기반으로 협찬/광고성 리뷰 감지 및 분리 |
| 단점 중심 큐레이션 | 단점을 상세히 기술한 리뷰를 우선 노출하여 균형 잡힌 정보 제공 |
| 카테고리별 개인화 | 분야 특성에 맞는 개인화 기준 적용 |
| 가격 비교 및 구매 연결 | 플랫폼별 가격 표시 및 구매 링크 제공 |

## 3. 사용자 정의

### 3.1 타겟 사용자

#### 주요 페르소나

**페르소나 1: 신중한 구매자 (민지, 28세)**

| 항목 | 내용 |
|------|------|
| 직업 | 마케팅 대행사 근무 |
| 쇼핑 빈도 | 월 2~3회 온라인 구매 |
| 특징 | 구매 전 최소 20개 이상 리뷰 확인, 단점 위주로 탐색 |
| Pain Point | 광고성 리뷰와 진짜 리뷰 구분이 어려움, 여러 플랫폼을 돌아다녀야 함 |
| 니즈 | 광고 없는 솔직한 리뷰만 빠르게 확인하고 싶음 |
| 기대 행동 | 상품 검색 → 광고성 필터 적용 → 단점 중심 리뷰 확인 → 구매 결정 |

**페르소나 2: 사이즈 고민러 (수진, 32세)**

| 항목 | 내용 |
|------|------|
| 직업 | 회계법인 근무 |
| 쇼핑 빈도 | 월 1~2회 |
| 특징 | 키 158cm, 체중 52kg으로 사이즈 선택에 어려움 |
| Pain Point | 브랜드마다 사이즈가 달라 자주 교환/반품 |
| 니즈 | 나와 비슷한 체형의 사람이 어떤 사이즈를 선택했는지 알고 싶음 |
| 기대 행동 | 체형 프로필 등록 → 유사 체형 리뷰 필터 → 사이즈 결정 |

**페르소나 3: 가성비 탐색러 (현우, 35세)**

| 항목 | 내용 |
|------|------|
| 직업 | IT 기업 개발자 |
| 쇼핑 빈도 | 분기 1~2회 |
| 특징 | 쇼핑에 시간 투자 최소화, 가격 대비 품질 중시 |
| Pain Point | 같은 상품인데 플랫폼마다 가격이 달라 비교하기 번거로움 |
| 니즈 | 가격 비교와 리뷰를 한 번에 확인하고 최저가로 구매하고 싶음 |
| 기대 행동 | 상품 검색 → 가격 비교 → 리뷰 요약 확인 → 최저가 플랫폼에서 구매 |

### 3.2 사용자 시나리오

#### 시나리오 1: 광고 없는 진짜 리뷰 찾기

```
상황: 민지(28세)가 새 코트를 구매하려고 함

1. 민지가 서비스에 접속
2. 검색창에 "로에일 오버핏 코트" 입력
3. 검색 결과에서 해당 상품 클릭
4. 상품 상세 페이지에서 "광고성 리뷰 제외" 필터 적용
5. "단점 언급 리뷰" 정렬 선택
6. 단점을 상세히 기술한 리뷰들 확인
   - "보풀이 좀 생겨요", "팔이 좀 짧은 편"
7. 장단점 파악 후 구매 결정
8. 가격 비교 확인 후 최저가 플랫폼 링크 클릭
```

#### 시나리오 2: 체형 기반 사이즈 결정 (패션 특화)

```
상황: 수진(32세)이 처음 서비스를 이용

1. 수진이 서비스에 회원가입
2. 체형 프로필 설정 화면으로 이동
3. 키(158cm), 몸무게(52kg), 평소 사이즈(S~M) 입력
4. 관심 있는 니트 상품 검색
5. 리뷰 목록에서 자동으로 유사 체형 리뷰가 상단에 정렬됨
6. "비슷한 체형" 배지로 빠르게 관련 리뷰 식별
7. 사이즈 인사이트 확인: "비슷한 체형 80%가 M 선택"
8. 리뷰 내용 확인 후 사이즈 결정
```

#### 시나리오 3: 가격 비교 후 최저가 구매

```
상황: 현우(35세)가 점심시간에 빠르게 바지 구매 결정

1. 현우가 서비스 접속
2. 상품 URL 붙여넣기로 빠른 검색
3. 상품 상세 페이지 진입
4. 플랫폼별 가격 비교 확인
   - 29cm: 45,000원
   - 무신사: 42,000원 ← 최저가
   - 네이버: 44,000원
5. 리뷰 요약에서 핵심 단점 확인: "허리가 타이트함"
6. 무신사 최저가 링크 클릭하여 구매 진행
```

**기능 ID**: F-001  
**우선순위**: P0

##### 기능 설명
사용자가 서비스에 가입하고 로그인할 수 있는 기능

##### 상세 요구사항

| ID | 요구사항 | 필수 여부 |
|----|----------|----------|
| F-001-01 | 이메일/비밀번호로 회원가입 가능 | 필수 |

| F-001-07 | 로그인 상태 유지 (7일) | 필수 |
| F-001-08 | 비밀번호 찾기 (이메일 인증) | 필수 |

패션 카테고리 전용. 체형 정보 기반 사이즈 추천 및 통계 제공.

##### 상세 요구사항

| ID | 요구사항 | 필수 여부 |
|----|----------|----------|
| F-007-01 | 유사 체형 기반 사이즈 추천 | 필수 |
| F-007-02 | 사이즈별 구매 비율 차트 | 필수 |
| F-007-03 | 사이즈 피드백 분포 (작음/적당/큼) | 필수 |
| F-007-04 | 추천 신뢰도 표시 (리뷰 수 기반) | 필수 |
| F-007-05 | 프로필 미등록 시 전체 평균 데이터 표시 | 필수 |

##### 수락 기준

- [ ] 사용자 프로필(165cm, 55kg) 등록 후 상품 조회 → "비슷한 체형 N%가 M 선택" 표시
- [ ] 사이즈별 구매 비율 → 가로 막대 차트로 표시 (S: 10%, M: 60%, L: 30%)
- [ ] 사이즈 피드백 분포 → "작음 20% / 적당 65% / 큼 15%" 표시
- [ ] 유사 체형 리뷰 3개 미만 → 신뢰도 "낮음" 표시
- [ ] 유사 체형 리뷰 10개 이상 → 신뢰도 "높음" 표시
- [ ] 프로필 미등록 → 전체 평균 데이터 표시 + "프로필 등록 시 맞춤 추천" 유도

##### 체형 유사도 계산 기준

| 항목 | 기준 |
|------|------|
| 계산 공식 | `distance = sqrt((height_diff/10)^2 + (weight_diff/5)^2)` |
| 유사 판정 | distance < 5 (약 ±5cm, ±2.5kg 범위) |
| 신뢰도 기준 | 유사 체형 리뷰 3개 미만: 낮음, 3~9개: 보통, 10개 이상: 높음 |

##### 엣지 케이스

| 상황 | 처리 방안 |
|------|----------|
| 유사 체형 리뷰 0개 | "비슷한 체형의 리뷰가 아직 없습니다" 안내 + 전체 평균 데이터 표시 |
| 리뷰에 체형 정보 없음 (대부분) | 체형 정보가 있는 리뷰만으로 통계, 부족 시 "데이터 부족" 안내 |
| 체형 정보가 비정상적 (키 300cm 등) | 수집 시 유효 범위(100~220cm, 30~200kg) 밖 데이터 제외 |

## 10. 용어 정의

| 용어 | 정의 |
|------|------|
| 진성 리뷰 | 협찬, 광고, 체험단이 아닌 실제 구매자의 솔직한 후기 |
| 광고성 리뷰 | 협찬/제공/체험단/광고 목적으로 작성된 리뷰 |
| 단점 상세도 | 리뷰 내 단점 언급의 구체성 정도를 수치화한 점수 |
| 통합 리뷰 | 여러 플랫폼에서 수집한 리뷰를 하나의 목록으로 합친 것 |
| 가격 비교 | 동일 상품의 플랫폼별 가격을 비교하여 표시하는 기능 |
| 체형 유사도 | 사용자의 키/몸무게와 리뷰어의 키/몸무게를 비교하여 계산한 유사성 점수 (0~100%) |
| 사이즈 피드백 | 리뷰어가 구매한 사이즈에 대한 평가 (작음/적당/큼) |
| 사이즈 인사이트 | 리뷰 데이터를 분석하여 도출한 사이즈 관련 통계 및 추천 |
| 비슷한 체형 배지 | 체형 유사도 70% 이상인 리뷰에 표시되는 시각적 표시 |
| 버티컬 | 서비스가 다루는 특정 상품 카테고리 (패션, 뷰티, 가전 등) |
| 어필리에이트 | 구매 링크를 통한 판매 수수료 수익 모델 |
| MVP | Minimum Viable Product, 최소 기능 제품 |
| MAU | Monthly Active Users, 월간 활성 사용자 수 |
| DAU | Daily Active Users, 일간 활성 사용자 수 |
